emotionのインストール

すでに顔識別のインストールされた状態で
V_face 仮想環境に入り

pip install deepface
pip install tf-keras

の2つをインストールしてください。
少しワーニングがでますが、気にせずに...

インストールできたら
emo_camera.pyとemo_photo.pyをfaceディレクトリにコピーしてください。

実行
python emo_camera.py カメラの画像から感情分析します。
python emo_photo.py  保存された画像から感情分析します。
                     ファイルは固定なので、プログラムを見てください。

emo_camera.pyの使い方は、headshots_OnePerson.pyとほぼ同じです。


使ってみての感想
・口角を上げて、少し歯を見せると喜びのスコアが上がります。
・普通にしていると、中立のスコアが高くなりますね。
・その他のスコアを意図的に上げようとしてもうまく表情が作れない
